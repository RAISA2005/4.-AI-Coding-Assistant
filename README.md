# 💻 AI Coding Assistant using LangChain, Ollama & Streamlit

## 📌 Project Description

This project is an AI-powered Coding Assistant developed using Python, LangChain, Ollama, and Streamlit. The application provides domain-specific assistance for programming and computer science topics, including Python, SQL, Machine Learning, Data Science, and General Computer Science concepts.

The assistant uses prompt engineering and a locally hosted Large Language Model (LLM) through Ollama to generate accurate responses while restricting answers to the user-selected domain. The application is deployed using Streamlit, providing an interactive and user-friendly web interface.

---

## 🎯 Project Objectives

* Build an AI-powered coding assistant using a Large Language Model (LLM).
* Provide domain-specific coding assistance.
* Restrict responses based on the selected programming domain.
* Demonstrate prompt engineering using LangChain.
* Deploy the application through Streamlit for real-time interaction.

---

## 📂 Project Structure

The project consists of three major components:

### 1. Prompt Engineering

### 2. AI Response Generation

### 3. Streamlit Web Application

---

# 🧠 Prompt Engineering

The application uses LangChain to design structured prompts for the Large Language Model.

### Prompt Features

* Domain-specific instructions
* Programming language selection
* Response restriction
* Stateless conversation
* Controlled AI behavior

The prompt ensures the assistant only answers questions related to the selected domain.

---

# 🤖 AI Response Generation

The application uses a locally hosted Ollama model to generate responses.

### Supported Domains

* Python
* SQL
* Machine Learning
* Data Science
* General Computer Science

### Response Process

* User enters a coding question.
* The selected domain is included in the prompt.
* LangChain processes the request.
* Ollama generates an AI response.
* The answer is displayed instantly.

---

# 🌐 Streamlit Web Application

Streamlit provides an interactive interface for users.

### Features

* Domain selection sidebar
* Interactive coding question input
* Real-time AI-generated responses
* Stateless interaction (no conversation memory)
* Clean and responsive interface

### Workflow

1. User selects a domain.
2. User enters a coding-related question.
3. LangChain prepares the prompt.
4. Ollama generates the response.
5. The answer is displayed in the application.

# ⚙️ Installation & Setup

### Clone the Repository

```bash
git clone https://github.com/RAISA2005/AI_Coding_Assistant.git
```

### Navigate to the Project Directory

```bash
cd AI_Coding_Assistant
```

### Install Required Dependencies

```bash
pip install -r requirements.txt
```

### Run Ollama

Make sure Ollama is installed and the required model is available locally.

### Run the Streamlit Application

```bash
streamlit run app.py
```
